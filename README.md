# Expense Tracker – Java Project

## Overview
A simple Java console-based application to record and analyze daily expenses. The project stores data using file handling and provides useful summary reports like total spending, category breakdown, and highest spending day.

## Features
- Add expenses with date, category, and amount  
- Calculate total expenses  
- Category-wise spending summary  
- Identify highest spending day  
- View all stored expenses  
- Data stored in a text file for persistence  

## Technologies Used
- Java  
- File Handling  
- OOP Concepts (Classes, Objects, Encapsulation)

## Project Structure
ExpenseTrackerProject/
 ├── ExpenseTracker.java
 ├── Expense.java
 └── expenses.txt

*(expenses.txt is auto-created if missing)*
